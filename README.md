# Catálogo de Produtos

Projeto simples feito em React para exibir um catálogo de produtos.

## Funcionalidades

- Exibe produtos na tela
- Permite adicionar novos produtos
- Valida campos do formulário
- Simula carregamento com useEffect

## Tecnologias usadas

- React
- Vite
- JavaScript

## Como rodar

1. Instalar dependências:
npm install

2. Rodar o projeto:
npm run dev
